Do This First
Before anything else — before uploading documents, before filling out forms — make this one phone call. It takes 3 minutes and protects every dollar you're about to earn.
⏰
This is called an Intent to File (ITF). Once you make this call, the VA locks in today's date as the start of your compensation. Every day you wait is a day of backpay you can never recover.
1-800-827-1000
Say exactly this: "Hello, I would like to submit an Intent to File for compensation, and I need to note on my record that I am currently experiencing a hardship."
The agent will be professional and helpful. They only need your name and Social Security number. This call locks in your start date immediately.
I made the call and submitted my Intent to File
